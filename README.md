# UJN-FinanceMathematics

> 济南大学金融数学专业课程试卷与复习资料整理仓库
> A collection of exam papers and review materials for Finance Mathematics at University of Jinan.

## 📌 仓库简介

本仓库用于整理和归档济南大学金融数学专业相关课程的历年试卷、复习资料、课程笔记以及部分学习参考内容，方便同学们进行期末复习、课程回顾和知识体系梳理。

仓库内容主要面向金融数学专业本科阶段课程，涵盖数学基础课程、金融类课程、统计与计量课程、计算机相关课程等。

仓库计划按照课程或模块进行整理，目录结构示例如下：

## 📝 资料类型

本仓库可能包含以下类型的资料：

* 历年期末试卷
* 课程复习提纲
* 课后习题与答案
* 课堂笔记整理
* 名词解释与简答题整理
* 计算题题型总结
* 课程论文或实验报告模板

## ✅ 使用说明

你可以通过以下方式使用本仓库：

1. 根据课程名称查找对应文件夹；
2. 优先查看历年试卷，了解考试题型；
3. 结合复习提纲整理知识框架；
4. 对照习题与答案进行查漏补缺；
5. 如发现错误或资料缺失，欢迎提交 Issue 或 Pull Request。

## 🤝 贡献指南

欢迎同学们共同完善本仓库。

如果你想贡献资料，可以按照以下建议提交：

1. 尽量使用清晰的文件命名方式；
2. 文件名建议包含课程名、年份、资料类型；
3. 上传前请尽量去除个人隐私信息；
4. 如果资料来源不确定，请在文件说明中注明；
5. 如果发现已有资料存在错误，可以提交修改说明。

推荐文件命名格式：

```text
课程名称-年份-资料类型.pdf
```

示例：

```text
金融工程-2024-期末试卷.pdf
金融风险管理-2025-复习提纲.pdf
计量经济学-2023-课后习题答案.pdf
```
